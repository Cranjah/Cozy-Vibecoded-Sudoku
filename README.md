## COZY VIBECODED SUDOKU

### Description of the Functionalities:
In here you will find a cozy vibecoded sudoku generator and solver, written in Python. Start the program by switching with your terminal into the "sudoku" folder and run the command "python3 gui.py" for macOS after installing Python onto your system and taking care about possible dependencies. After this, the Sudoku generator's and solver's Graphical User Interface (GUI) should pop up, ready for play. Completely vibecoded with Microsoft Copilot.
